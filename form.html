About the webpage

The webpage contains information and details on how a form is. The html text used here are text, inputs, labels, select, placeholders, values. All of which allwo the user to input data. The labels make it easy to identify what the input is for while the text will identify what data needs to typed in as the input by the user.

  1. Labels: are used to identify what is the input type is
  2. Input: was used to allow the user to input data
  3. Text: This would imply what the user is typing in. (It would also act as a form of data validation just in case they type in number instead of text)
  4. Select: was used for drop down lists
  5. Placeholder: are used after id tags in the input places and they act as directors for the users to know what to type.
