Css Basics

CSS stands for Cascading Style Sheets. It is the language for describing the presentation of Web pages, including colours, layout, and fonts, thus making our web pages presentable to the users. CSS is designed to make style sheets for the web. It is independent of HTML and can be used with any XML-based markup language. The three types of CSS include Inline CSS, Internal CSS, and External CSS. Inline CSS can be included directly in the HTML elements. Internal CSS is the other way of including CSS by using the style element in the head section of the HTML document. External CSS includes CSS by using an external stylesheet.
